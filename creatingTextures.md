# *Creating textures*
* > Get __pixel data__ from an array
* >Create an __ID__
* >Assign __Parameters__
* > OpenGL knows what image it's uploading
* > __Upload__ to GPU

# *Draw into an image*
* > Assign UV coordinates
* > Have some uniform variables 
* > GL gives a special case allowing to choose textures we uploaded
* > __BIND__ the texture to a slot
* > __Shader__ will get the data from that slot and apply the texture into the UV
